# Нумерологическая матрица (Квадрат Пифагора)

Программа на C++ для автоматического расчёта нумерологической матрицы (психоматрицы) по дате рождения. Алгоритм полностью воспроизводит классическую методику расчёта рабочих чисел и подсчёта количества цифр в итоговой строке.

## Особенности реализации

*   **Полная совместимость со старыми компиляторами:** Код написан в стандарте C++98. Это позволяет запускать его даже на старых версиях MinGW (как в Dev-C++ 4.9.2), избегая ошибок вроде `non-static data member initializers only available with -std=c++11`.
*   **Отсутствие зависимости от `std::to_string`:** Для конвертации чисел в строки используется собственная функция `intToString`, что гарантирует работу без включения новых стандартов C++.
*   **Защита от ошибок ввода:** Программа проверяет формат даты (обязательно `XX.XX.XXXX`). При неверном вводе программа корректно завершается с сообщением об ошибке, а не падает.
*   **Локализация:** Поддерживается вывод текста на русском языке (требуется консоль с соответствующей кодировкой).

## Алгоритм расчёта

Программа выполняет следующие шаги:

1.  **Парсинг даты:** Разбивает строку `XX.XX.XXXX` на день, месяц и год.
2.  **Расчёт `num1`:** Сумма всех цифр даты.
3.  **Расчёт `num2`:** Сумма цифр числа `num1`.
4.  **Расчёт `num3`:** Формула `num1 - 2 * pr`, где `pr` — первая цифра дня рождения.
5.  **Расчёт `num4`:** Сумма цифр числа `num3`.
6.  **Формирование строки:** Склеивает дату и все 4 рабочих числа в одну длинную строку.
7.  **Подсчёт:** Анализирует строку и считает количество вхождений каждой цифры от 1 до 9.
